## Sous-phases

### v0.7a — Python Binding (PyO3) (3-4 jours)

- [ ] `bindings/python/` : package PyO3
- [ ] `Cargo.toml` : feature `pyo3`, `dep:pyo3`
- [ ] Exposer HoraCore comme classe Python
- [ ] Mapping des types :

| Rust | Python |
|------|--------|
| `EntityId(u64)` | `int` |
| `&str` | `str` |
| `HashMap<String, PropertyValue>` | `dict` |
| `&[f32]` embedding | `numpy.ndarray` ou `list[float]` |
| `Result<T>` | Exception `HoraError` |
| `Option<T>` | `T \| None` |

- [ ] Support numpy pour les embeddings (zero-copy si possible)
- [ ] `pip install hora-graph-core` (maturin build)
- [ ] Type stubs `.pyi` pour autocompletion

**API Python :**
```python
from hora_graph_core import HoraCore

hora = HoraCore.open("memory.hora", embedding_dims=384)
entity_id = hora.add_entity("project", "hora", {"language": "Rust"})
results = hora.search("authentication", top_k=10)
hora.dream_cycle()
hora.close()
```

**Tests :**
- CRUD complet depuis Python
- Embeddings numpy → vector_search
- Properties dict → recuperables
- Exception HoraError correcte
- Persistence : open → add → flush → close → reopen → verify

### v0.7b — WASM Binding (2-3 jours)

- [ ] `bindings/wasm/` : package wasm-bindgen
- [ ] Feature `wasm` : `dep:wasm-bindgen`
- [ ] Restrictions WASM :
  - Pas de mmap → fallback read()
  - Pas de multi-thread → single-threaded OK
  - Pas de filesystem → Memory backend uniquement (ou IndexedDB futur)
  - SIMD 128-bit disponible (pas AVX2)
- [ ] Exposer HoraCore comme classe JS (browser)
- [ ] Float32Array pour les embeddings

**API WASM :**
```javascript
import init, { HoraCore } from 'hora-graph-core-wasm';
await init();

const hora = HoraCore.newMemory({ embeddingDims: 384 });
const id = hora.addEntity('project', 'hora', { language: 'Rust' });
const results = hora.search('authentication', null, { topK: 10 });
hora.close();
```

**Tests :**
- CRUD complet en WASM (wasm-pack test)
- Vector search avec WASM SIMD
- Memory backend uniquement
- Performance : 3-5x plus lent qu'en natif = acceptable

### v0.7c — C FFI Header (1-2 jours)

- [ ] `bindings/c/hora.h` : header C genere
- [ ] ABI stable : `extern "C"` functions
- [ ] Opaque pointer : `typedef struct HoraCore HoraCore;`
- [ ] Error handling : return codes + `hora_last_error()` string
- [ ] Memory management : `hora_free_*()` pour chaque type retourne

**API C :**
```c
#include "hora.h"

HoraCore* hora = hora_open("memory.hora", NULL);
uint64_t id = hora_add_entity(hora, "project", "hora", NULL, NULL);
HoraEntity* entity = hora_get_entity(hora, id);
hora_free_entity(entity);
hora_close(hora);
```

**Tests :**
- C test binaire qui linke avec libhora
- CRUD complet
- Error handling : codes retour corrects
