## Post-v1.0 : roadmap

### v1.1 — Performance
- HNSW optimise (product quantization, re-ranking)
- CSR page-based pour les edges (au lieu du B+ tree)
- Parallel dream cycle (rayon)

### v1.2 — Features
- Embedding integre optionnel (feature `embedder` + ONNX models)
- Graph algorithms : PageRank, community detection
- Subscription/watch sur des queries

### v1.3 — Ecosystem
- IndexedDB backend pour WASM persistant
- REST API server optionnel
- Visualization tool (graphe interactif)
- VS Code extension

### v2.0 — Breaking (si necessaire)
- Format .hora v2 avec compression zstd
- MVCC complet au lieu de SW/MR
- Distributed mode (Raft consensus)
